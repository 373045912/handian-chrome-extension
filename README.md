汉典划词查询的Chrome插件
========================

# 这是什么

汉典（ http://www.zdic.net/ ）是非常优秀的文言文查询网站，本插件让你在任何页面内能够以划词的方式使用汉典进行查询。

# 使用方法

1. 在download文件夹下选择版本号最大的文件下载
2. 打开Chrome，菜单->工具->扩展程序
3. 将已下载的crx文件拖入Chrome窗口
4. 现在你应该能看到有个“典”字的图标了，点击它，当你需要开启划词查询的时候选中“是否启用”
